grubfs
======

Install dependencies:

```
npm install
npm install -g grunt-cli
npm install -g browserify
```

Install unittest dependencies:
```
npm install -g mocha
```


Run jshint and build client-side script:

```
grunt
```

Start development server:

```
npm run-script dev
```

View server at localhost:8080.

Run unittests:

```
mocha
```
